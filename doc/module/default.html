Mail: whoami [AT] dev-urandom [DOT] eu

DefaultModule

This module handles the default communication with the IRC server, like registering on the server after the connection has been opened

Configuration

gbt.conf 

    {"Config":{
         "CmdPrefix":"&",
         "Nickname":"gbt_dev",
         "Realname":"gbt",
         "Username":"gbt"
    }}
The CmdPrefix is the prefix of that is used for bot command in channels

Commands

&whoami
Tells you who you are
&help [COMMAND]
When invoked without a COMMAND it lists all available commands otherwise it shows the help for COMMAND

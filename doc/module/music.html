GBT - a irc bot written in go

MusicModule

Different music related function. At the moment only current tracks on last.fm are implemented.

Configuration

gbt.conf

{"Config":{
    "lfm_default":"AlphaBernd",
    "lfm_map":{
        "AlphaBernd":"rj"
    }
 }}

lfm_default is the default nick for last.fm queries. lfm_map maps irc nicknames to last.fm nicknames.

Commands

&np [NICK]
Print the last scrobbled song from NICK. If NICK is not given the default nick is used.
&lfm.total [NICK]
Print the total number of songs NICK has listened to

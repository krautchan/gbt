FortuneModule

This Module posts quotes from the fortune database to a channel

Configuration

fortune.conf

{"Config":{
    "db-path":"/usr/share/games/fortunes"
}}

If run is set to false the module won’t run.

Commands

&fortune [CATEGORY]
Send fortune to channel if CATEGORY is given the fortune will be from that category
&fortune.list
List all available categories

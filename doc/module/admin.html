AdminModule

This module authenticates the admin user and offer command for general bot control

Configuration

admin.conf:

{"Config":
    {"password":"XXXXXXXXXXXXX"}
}

The admin password should be changed before using the bot

Commands

&identify PASSWORD
Authenticate yourself to the bot with PASSWORD
&nick NICKNAME
Change the nickname of the bot to NICKNAME[Authentication required]
&join CHANNEL
Tell the bot to join CHANNEL[Authentication required]
&part CHANNEL
Tell the bot to leave CHANNEL[Authentication required]

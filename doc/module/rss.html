RSSModule

This Module fetches RSS feeds and sends them to the IRC server. It can also create a .rss news file that can be served by any http server

Configuration

rss.conf 

    {"Config":{
        "feeds":{
            "fefe":"http://blog.fefe.de/rss.xml",
            "test":"http://wiki.dev-urandom.eu/news.rss"
        },
        "news.file":"news.xml",
        "news.url":"http://wiki.dev-urandom.eu/news.rss"
    }}
All settings can be changed with commands. The two entries in feeds are shortcurt commands to fetch the named feeds. news.file sets the file to where to save news to. news.url is the url under which the news file is reachable

Commands

&rss
Print the first items from the given RSS feed
&rss.add COMMAND URL
Add shortcut COMMAND for rss[Authentication required]
&rss.del COMMAND
Delete shortcut COMMAND[Authentication required]
&rss.list
Show available shortcut commands
&news [NEWS]
Show the latest news or post a new NEWS item
&news.setFile FILEPATH
Set file where news should be saved[Authentication required]
&news.setUrl URL
Set URL to where to read news from[Authentication required]
